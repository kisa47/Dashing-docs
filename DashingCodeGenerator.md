# Программа DashingCodeGenerator

[Ссылка на исходный код](https://github.com/kisa47/DashingCodeEditor)
[Скачать для Android](https://github.com/kisa47/DashingCodeEditor/releases/latest)

Программа служит для просмотра текущей конфигурации или её изменения на основе кодировок комплектаций в блоках автомобиля.

## Руководство

Для начала нужно выбрать тип блока, изменения кодировки которого планируется произвести.

Далее необходимо ввести (скопировать) кодировку блока полученную сканером (например Launch) или выбрать из предложенных.

Далее в списке функций производим изменения кодировки. В случае необходимости можно отменить изменения нажав кнопку **Reload**. При долгом нажатии на имя функции всплывёт подсказка.

Для генерации изменённоё кодировки нажимаем кнопку **Generate**. Те символы, которые изменились относительно исходной кодировки будут отмечены цветом.

Для записи кодировки в блок сканером, можно скопировать изменённую кодировку кнопкой **Copy**.
